This application will be a web application that allows users to track their life totals in a multiplayer game of Commander(EDH) in Magic: The Gathering. It is targeted for use with a tablet computer sitting in the middle of the table. Players will use the same tablet to track their life totals. 

# User Interface

- Configurable for 2-6 players
- Each player has their own life total
- The life totals are displayed in a grid where the each life total is oriented towards the closest edge of the screen
- Life totals are adjusted by +1 or -1 using touch input on the life total display
- The life total display is large and takes up most of the screen
- There is a thin menu bar at the top of the screen that allows for:
  - adding a player
  - removing a player
  - randomly selecting a player
  - resetting the life totals

# Other Functionality

- The application should be able to be used offline
- The application should be able to be used on a tablet or phone
- The application will automatically save the state of the game and reload it from local storage
- the application will be a single page progressive web app

# Implementation stack

- Typescript
- React
- Vite
- Tailwind
